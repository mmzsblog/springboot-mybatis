## springboot整合mybatis

- 整合后连接数据库做访问测试
- 本地启动后访问：[http://localhost:10008/single/test](http://localhost:10008/single/test)

- 整合多数据源后连接数据库做访问测试

- 本地启动后访问从数据库数据：[http://localhost:10008/multiple/slave](http://localhost:10008/multiple/slave)
